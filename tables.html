HTML5 Table Examples

This page demonstrates various capabilities and styling options for HTML5 tables using CSS.

Table Example 1: CSS Attributes

Demonstrates setting table attributes (width, cellpadding, cellspacing, alignment, border, bgcolor) using CSS properties.

CSS Table Attributes in Action
Attribute Values / Format
Width e.g., `width: 80%;` or `width: 600px;`
Padding (cellpadding) e.g., `padding: 10px;` on `td, th`
Spacing (cellspacing) e.g., `border-collapse: collapse;` on `table` (removes spacing)
Alignment e.g., `text-align: center;` on `td, th` or `margin: auto;` on `table`
Border e.g., `border: 2px solid #948979;` on `table` and `border: 1px solid #393E46;` on `td, th`
Background Color (bgcolor) e.g., `background-color: #222831;` on `table` or `td`
Number of Rows/Columns This table has two (2) columns x seven (7) rows.
